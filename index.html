Paramètres de Synchronisation Survey123

Cette plateforme se connecte en temps réel aux données collectées sur le terrain via l'application mobile Survey123. Les réponses sont enregistrées dans un service d'entités (Feature Layer) hébergé sur ArcGIS Online.

🗝️ Comment se connecter sans jeton (Token) ?

Si votre Survey123 est configuré comme public, vous n'avez pas besoin de jeton. Il vous suffit de saisir l'URL du service ci-dessous et de laisser le champ jeton vide. Pour rendre votre Survey123 public :

  1. Connectez-vous sur survey123.arcgis.com.
  2. Ouvrez votre enquête, allez dans l'onglet Collaborer.
  3. Dans Partager l'enquête et Partager les données, cochez Tout le monde (public).
  4. Alternativement : Dans ArcGIS Online, recherchez la couche d'entités associée (Feature Layer) et modifiez son partage en Public.

🔒 Méthode alternative (Couche Privée avec Jeton)

Si vous préférez garder vos données privées, vous devez générer un jeton temporaire :

  1. Allez sur le portail : ArcGIS Generate Token Portal
  2. Saisissez l'URL du service (présente ci-dessous) et générez le jeton.
  3. Copiez-le et collez-le dans le champ "Jeton d'accès (Token)" ci-dessous.
